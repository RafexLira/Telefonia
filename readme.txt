API com planos e servicos de telefonia. API Básica para testes onde é possível consultar, incluir e deletar a operadora, os planos suportados e os serviços oferecidos. Para se ter acesso basta digitar o host seguido do endereço http, exemplo: localhost:44379/api/Operadora - Para listar todas as operadoras localhost:44379/api/Plano - Para ter acesso aos planos de cada operadora localhost:44379/api/Servico - Listar os servicos oferecidos vinculados ao tipo de plano

Caso queira acessar uma operadora, plano ou servico específico basta digitar o seu Id ao final do link, exmeplo: localhost:44379/api/Operadora/5

Foi feito no visual studio em C# .NET FrameWork com Entity e SQL Server.
